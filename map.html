Data Viz Project: Berkeley Crime Rates

Tiffany Lee, Rachel Li, Allison Xu | Web Dev Spring 2019

For our data visualization project, we chose to visualize crime in Berkeley using daily crime logs from UCPD for January through March 2019. We chose this because crime is an important issue that affects everyone’s day-to-day lives, and we want to be able to provide more information about it in an accessible and easy to understand manner.
